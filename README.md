# Project Name
HarmonyNow

# Project Description
HarmonyNow는 화성학 교육 서비스입니다.

화성학이란 소리를 어떻게 배치하면 잘 어울릴 수 있는지 탐구하는 음악 이론으로서, 작곡이나 편곡을 잘할 수 있게 도와줍니다.

HarmonyNow는 음악에 대한 기초 지식이 있는 초급/중급 학습자가 코드(chord) 중심으로 화성학을 배울 수 있도록 합니다.
